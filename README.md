# StepUp

StepUp is a fitness and workout application developed in Python using Pygame as part of the Hackathon (Nitzanim).

The application allows users to complete workout routines, track progress, and earn in-app coins that can later be exchanged for real rewards.

## Features

- User login and registration system
- Persistent user data storage using JSON
- Workout system with:
  - Exercise timers
  - Break timers
  - Exercise images/examples
- Coin and reward system
- Shop interface
- Multiple screens and UI management
- Custom UI components built from scratch

## Technologies Used

- Python
- Pygame
- JSON data storage
- Object-Oriented Programming (OOP)

## Development Process

StepUp was developed during the 2025 hackathon as part of the "Nitzanim" program.

The project included collaborative development, task division between team members, product planning, and building a complete working application from the initial idea to the final result.

## Installation

1. Clone the repository

2. Install dependencies:

```bash
pip install -r requirements.txt
```
3. Run the project:
```bash
python main.py
```
